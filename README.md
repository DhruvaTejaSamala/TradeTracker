#### Stock Trading Web Application

**Overview:** Developed a comprehensive stock trading web application featuring a user-friendly interface for tracking and managing stock portfolios. The application allows users to create accounts, log in securely, and view real-time stock data. The platform integrates a React frontend with a .NET backend, providing a robust and scalable solution for stock trading enthusiasts.

**Key Features:**

*   **User Authentication:** Implemented secure user authentication and authorization using JWT Bearer tokens.
*   **Stock Portfolio Management:** Enabled users to search for stocks, add them to their portfolio, and view detailed information about each stock.
*   **Real-Time Data Integration:** Leveraged Axios for fetching and displaying the latest stock data, ensuring users have access to up-to-date market information.
*   **Interactive UI:** Utilized Material-Tailwind for a modern and responsive UI, and Recharts for interactive data visualization.
*   **Notifications:** Integrated React Toastify for real-time notifications and alerts.
*   **API Documentation:** Employed Swashbuckle to generate comprehensive Swagger documentation for the backend API.

**Technologies Used:**

*   **Frontend:** React, React Router, Axios, React Hook Form, Yup, Material-Tailwind, Tailwind CSS, Recharts, React Toastify
*   **Backend:** .NET Core, Entity Framework Core (SQL Server), JWT Bearer Authentication, Swashbuckle, Microsoft Identity
*   **Database:** SQL Server
*   **Tools:** Visual Studio, Postman, Git

**Responsibilities:**

*   Designed and implemented the frontend using React and Material-Tailwind for a responsive and intuitive user experience.
*   Developed and secured backend APIs with .NET Core, ensuring efficient and secure data transactions.
*   Managed state and form handling in the frontend using React Hook Form and Yup.
*   Created and maintained SQL Server databases, including schema design and query optimization.
*   Integrated real-time stock data using Axios, ensuring the frontend displays the most current information.
*   Implemented authentication and authorization mechanisms using JWT tokens to secure API endpoints.
*   Generated and maintained API documentation using Swashbuckle, facilitating easier integration and testing.
*   Conducted thorough testing and debugging to ensure the application’s reliability and performance.

**Outcome:** Delivered a high-performance stock trading platform that enhanced user experience through real-time data integration and an intuitive interface. The project demonstrated proficiency in full-stack development, modern web technologies, and best practices in software development.
